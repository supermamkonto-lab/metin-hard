# METIN HARD

Repozytorium projektu **METIN HARD LOG** — eksportera logów sprzedaży z panelu UCP serwera Metin2 ([projekt-hard.eu/ucp](https://projekt-hard.eu/ucp)).

## Gdzie jest aktywna aplikacja?

**[`MetinHardLog_v2.0/`](MetinHardLog_v2.0/)** — jedyna aktywnie rozwijana i wspierana wersja programu (GUI, v2.0).
Uruchamia się przez **`MetinHardLog_v2.0/!URUCHOM_PROGRAM.cmd`**. Pełna dokumentacja: [`MetinHardLog_v2.0/README_PL.md`](MetinHardLog_v2.0/README_PL.md) / [`MetinHardLog_v2.0/README_ENG.md`](MetinHardLog_v2.0/README_ENG.md).

## Pozostałe foldery

| Folder | Zawartość |
|---|---|
| `MetinHardLog_v2.0/` | Aktualna wersja aplikacji (jedyna wspierana) |
| `OLD_v1.0/` | Zarchiwizowana, starsza działająca wersja (CLI, menu tekstowe, eksport CSV+XLSX) — zachowana wyłącznie jako punkt odniesienia, nie jest rozwijana |
