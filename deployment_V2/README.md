# METIN HARD LOG
## Eksporter Logow Sprzedazy v1.0

### Jak uruchomic?

1. Kliknij dwukrotnie **START.cmd**
2. Wybierz **1 -- Instalacja** (tylko za pierwszym razem)
3. Wybierz **3 -- Konfiguracja** (wpisz login, haslo, PIN)
4. Wybierz **4 -- Pobierz logi**
5. Obserwuj prace programu -- pliki pojawia sie w folderze **output/**

### Co zobaczysz?

Program otworzy przegladarke i wykona wszystkie operacje na Twoich oczach:
- zaloguje sie do panelu UCP,
- znajdzie postacie na koncie,
- pobierze logi sprzedazy,
- zapisze wyniki do plikow Excel i CSV.

### Konfiguracja

Plik `config.toml` zawiera:
- dane logowania (login, haslo, PIN)
- folder na wyniki
- opcja `show_browser = true` (widoczna przegladarka) lub `false` (praca w tle)

### Pomoc

Wybierz opcje **2 -- Diagnostyka** z menu.
