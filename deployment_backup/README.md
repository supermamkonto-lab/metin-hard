# METIN HARD Log Extractor

## Jak uruchomić?

1. Kliknij dwukrotnie **START.cmd**
2. Wybierz **1 — Instalacja** (tylko za pierwszym razem)
3. Wybierz **3 — Konfiguracja** (wpisz login, hasło, PIN)
4. Wybierz **4 — Pobierz logi**
5. Pliki XLSX i CSV pojawią się w folderze **output/**

## Wymagania

- Windows 11
- Internet
- Konto UCP na projekt-hard.eu

## Pomoc

Wybierz opcję **2 — Diagnostyka** z menu.
Program sam sprawdzi co nie działa i powie jak to naprawić.
